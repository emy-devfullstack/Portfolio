Gerenciador de Querys SAP B1

Repositório com consultas SQL otimizadas para o banco de dados do SAP Business One, abrangendo fornecedores, produtos, vendedores, pedidos e compras. Facilita a extração de dados para análises estratégicas e decisões empresariais.

Tecnologias: SQL

Portal de Consultas

Portal para consulta rápida de status de CNPJ, com informações como razão social, situação cadastral, endereço e atividade econômica.

Tecnologias: HTML, CSS, JavaScript, Python

Automação CNPJ

Script em Python que consulta o status de atividade de CNPJs via API pública da Receita Federal e exporta os resultados para um arquivo Excel.

Tecnologias: Python

Automação CEP

Script em Python que consulta a API ViaCEP para buscar logradouro e tipo a partir de CEPs, com tratamento de erros e exportação dos resultados para um arquivo CSV.

Tecnologias: Python
